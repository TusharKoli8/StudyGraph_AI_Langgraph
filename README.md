# StudyGraph AI using Langgraph

An intelligent student assistant built with LangGraph that helps learners with programming, AI, data science, and technical subjects through agentic workflows.

## Features

- Personalized learning assistance
- Multi step reasoning using LangGraph
- Question answering for technical subjects
- Context aware conversations
- Resource and concept recommendations
- Expandable agent architecture

## Tech Stack

- Python
- LangGraph
- LangChain
- LLM APIs (Groq)
- Vector Database (optional)


## Use Cases

- Exam preparation
- Coding interview practice
- Concept explanations
- Learning roadmaps
- Subject specific doubt solving


## Run
---bash

python main.py
